## How to get listed

Dareplane spans many module repositories under the
[bsdlab](https://github.com/bsdlab) organisation. If you have contributed to any
Dareplane module and are not listed here, please open a pull request against
[this repository](https://github.com/bsdlab/Dareplane) adding your name, or let
a maintainer know.

We welcome contributions of all kinds — new modules, bug fixes, documentation,
and examples. See the
[README](https://github.com/bsdlab/Dareplane/blob/main/README.md) for an
overview of the project and the
[documentation](https://bsdlab.github.io/Dareplane/) for module guides.
